## Descrição da solução

O AgroMars Vision é um sistema de monitoramento para agricultura espacial, com foco em estufas inteligentes para cultivo em outro planeta. A solução utiliza visão computacional para analisar imagens de vegetais(nesse caso o tomate) e identificar seu estágio de maturação através das cores, auxiliando na decisão do momento ideal para a colheita.

Além da visão computacional, o sistema monitora sensores ambientais importantes para o cultivo, como temperatura, umidade, CO₂, O₂, radiação, pH da água e luminosidade. <br><br>
Os dados são comparados com faixas ideais de operação e alertas são soados caso algum parâmetro está fora dos limites recomendados.
